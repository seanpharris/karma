# Karma Art Prompt Packs

Paste-ready prompt packs for generating original, license-safe prototype art.

## Player Sprite v2

Use these to replace the temporary `32x32` engineer sheet with a cleaner
professional paper-doll standard:

- [`player-v2-generation-checklist.md`](player-v2-generation-checklist.md)
- [`player-v2-base-body-prompts.md`](player-v2-base-body-prompts.md)
- [`player-v2-paper-doll-layer-prompts.md`](player-v2-paper-doll-layer-prompts.md)
- [`player-v2-downed-carry-prompts.md`](player-v2-downed-carry-prompts.md)

Recommended first batch:

1. Base body idle/walk/sprint/interact.
2. Engineer outfit overlay for the same animations.
3. Hair overlay for the same animations.
4. Multi-tool overlay for idle/walk/interact.

Do **not** overwrite the current runtime player sheet until generated sheets pass
validation and the v2 runtime/compositor is wired.
