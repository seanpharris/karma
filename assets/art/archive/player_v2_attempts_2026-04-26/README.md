# Player v2 archived attempts — 2026-04-26

This folder keeps the generated player-v2 art attempts that were useful for learning but are no longer the active prototype path.

Archived here:

- early Gemini checker/chroma references,
- direction-batch pose references,
- regenerated 64px full-sheet attempts,
- strict walk-strip patches,
- extracted/generated engineer runtime candidates.

Why archived:

- inconsistent generated character identity across directions/frames,
- weak or incorrect walk stepping,
- tool/wrench contamination in movement rows,
- chroma/background cleanup issues,
- direction-slot confusion compared with the cleaner knight reference asset.

Current active prototype player visual remains `assets/art/sprites/generated/player_v2_knight_8dir_4row_reference.png`.

Current new standardization work is under `assets/art/sprites/player_v2/` using 32x64 cells:
