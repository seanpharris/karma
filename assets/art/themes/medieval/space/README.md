# Space Theme

A cozy, eerie sci-fi setting centered around space stations with themes of isolation, system repair, and survival.

## Visual Assets

- **tileset.png**: 16×16 tileset sheet for space-themed environments
  - Includes: station walls, floors, windows, panels, reactors, hydroponics, living quarters

## Generation Schema

See `docs/worldbuilding/space-generation.json` for the full generation schema and story rules.

## Theme Characteristics

- **Tone**: Cozy, absurd, socially reactive, occasionally dark, isolated, mysterious
- **World Size**: 1000×1000 tiles (16px each)
- **Core NPCs**: Up to 6
- **Factions**: Up to 3
- **Active Quests**: Up to 5

## Key Locations

1. **Station Hub** - Main social and operational center
2. **Engineering Bay** - Danger/repair/sabotage zone
3. **Habitat Ring** - Personal life, survival, emotional conflicts
