# Static Event Atlas Candidates

Generated with Gemini on 2026-04-29 for non-animated/static Karma art needs.

These sheets are not wired into runtime yet. Treat them as raw candidate atlases to slice, clean up, convert, or promote into the art manifest later.

## Static art needs identified

Static means single-view/single-sprite art that does not need animation frames like the player model.

High-value categories:

- Event signage: bounty boards, wanted notices, rumor boards, faction boards.
- Law/checkpoint props: scanner arch, barrier, law beacon, evidence bags.
- Supply/trade props: supply crates, trade tables, black market stalls, lockboxes.
- Rescue/clinic props: med kiosk, stretcher, triage station, quarantine marker.
- World-state props: damaged/repaired generators, gates, barricades, leaks, hatches.
- Player-to-player interaction props: contract markers, apology gifts, IOUs, trust/vouch seals, wager tables.
- Theme variant props: western, space/scifi, post-apoc, fantasy versions of the same underlying event mechanics.
- Environmental filler: decals, small rocks, cables, debris, grass, puddles, scorch marks.
