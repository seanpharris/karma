# Audio Assets

Temporary and curated runtime audio for Karma.

## Music

- `music/main_menu_theme_placeholder.wav` — original generated placeholder loop for the main menu. It is intentionally simple and safe to replace later with a composed theme.

## Curation Notes

- Keep third-party audio out of runtime folders unless the license is clear and documented.
- Prefer original/generated placeholders during prototyping.
- When external audio is used later, record source URL, author, license, and required attribution in this folder or `THIRD_PARTY_NOTICES.md`.
