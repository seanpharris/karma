# Boarding School Theme

A cozy, whimsical, socially tense boarding school setting with themes of secrets, gossip, relationships, and moral complexity.

## Visual Assets

- **tileset.png**: 16×16 tileset sheet for boarding school-themed environments
  - Includes: stone buildings, classrooms, dormitory rooms, courtyard, library, notice boards, furniture, school gardens

## Generation Schema

See `docs/worldbuilding/boarding_school-generation.json` for the full generation schema and story rules.

## Theme Characteristics

- **Tone**: Cozy, absurd, socially reactive, occasionally dark, gossipy, emotionally charged
- **World Size**: 1000×1000 tiles (16px each)
- **Core NPCs**: Up to 6
- **Factions**: Up to 3
- **Active Quests**: Up to 5

## Key Locations

1. **Courtyard** - Main social hub where gossip spreads
2. **Dorms** - Private life, secrets, friendships, and betrayal
3. **Classroom Wing** - Rules, authority, cheating, and pressure

## Key Roles

- Headmaster, teacher, top student, troublemaker, caretaker (required)
- Bully, shy student, gossip, prefect, rule follower, rule breaker, new student, secret club member, teacher's pet (optional)
