# Western Theme

A cozy, absurd, slightly dark western setting centered around frontier towns with themes of law, survival, and moral ambiguity.

## Visual Assets

- **tileset.png**: 16x16 tileset sheet for western-themed environments
  - Includes: wooden buildings, saloon, sheriff's office, mine entrance, farmhouse, desert terrain, fences, water troughs

## Generation Schema

See `docs/worldbuilding/western-generation.json` for the full generation schema and story rules.

## Theme Characteristics

- **Tone**: Cozy, absurd, socially reactive, occasionally dark
- **World Size**: 1000x1000 tiles (16px each)
- **Core NPCs**: Up to 6
- **Factions**: Up to 3
- **Active Quests**: Up to 5

## Key Locations

1. **Town Center** - Main social hub (saloon, sheriff office, general store, doctor office)
2. **Red Ridge Mine** - Danger/resource conflict zone
3. **Sunset Homestead** - Family, survival, and moral dilemma zone

## Key Roles

- Sheriff, doctor, saloon keeper, ranch owner, miner (required)
- Gambler, bandit scout, undertaker, preacher, shopkeeper, deputy, farmhand, runaway, traveling salesman (optional)
